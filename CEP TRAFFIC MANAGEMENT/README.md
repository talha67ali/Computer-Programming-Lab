# 🚦 TRAFFIC MANAGEMENT SYSTEM

## Computer Programming Lab (CEP)

---

# 📌 OVERVIEW

A simple C++ program that simulates traffic flow on 4 roads and assigns signal timing based on vehicle count.

---

# 🎯 OBJECTIVE

* Monitor traffic using vehicle input
* Analyze traffic level
* Assign signal timing automatically

---

# 🚗 LOGIC

* **LOW**: < 20 → 20 sec
* **MODERATE**: 20–50 → 40 sec
* **HEAVY**: > 50 → 60 sec

---

# 🖥️ FEATURES

✔ Menu-driven system
✔ 4-road simulation
✔ Traffic analysis
✔ Signal timing output

---

# 🚦 MENU

```
1. Enter Data  
2. Display Data  
3. Check Status  
4. Signal Timing  
5. Exit  
```

---

# 📚 CONCLUSION

A basic traffic simulation using C++ concepts like loops, arrays, functions, and conditions.
